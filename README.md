Deno :: Serverless :: AWS
---

#### Requirements
```
# Global installation of the serverless cli
npm install -g serverless

# Global installation of the serverless-scriptable plugin
npm install -g serverless-scriptable-plugin
```

deploy using `serverless deploy`
